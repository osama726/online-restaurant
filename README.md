# Online Restaurant

A production-style online restaurant backend built with Laravel, Docker, and modern software engineering practices.

## About

Online Restaurant is a RESTful API that enables customers to browse menus, place orders, track deliveries, and manage their accounts. The project is designed with scalability, clean architecture, and maintainability in mind.

This repository serves as a practical implementation of backend development best practices using the Laravel ecosystem.

---

## Tech Stack

- Laravel 12
- PHP
- MySQL
- Redis
- Docker
- REST API
